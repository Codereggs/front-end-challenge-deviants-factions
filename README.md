# Front-end NFT Game Challenge

This project is a react challenge made for an organization.

I have used many dependencies like:
-Next.js
-Typescript
-ChakraUI
-Redux-Toolkit

## Installation

Use the package manager to install the project.

```bash
yarn install
```

## Usage

```javascript
# when installation is ready use:
yarn start
```
